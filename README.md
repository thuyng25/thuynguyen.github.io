# thuyng25.github.io

My personal portfolio site, built to showcase my data analysis and visualization work while I job search for data analyst, BI, and operations roles.

**Live site:** [thuyng25.github.io](https://thuyng25.github.io/thuynguyen.github.io)

## About

I'm a recent Data Visualization grad from the University of Washington Bothell, with four years of operations experience running the numbers, inventory, and staffing side of a nails salon. This site pulls both sides of that together: a short bio, my featured projects, my skills, and a little interactive game (Spot the Outlier) that's really just a data instinct check in disguise.

## Built with

- HTML, CSS, and vanilla JavaScript (no framework, no build step)
- Google Fonts (Outfit, Inter, Caveat)
- Deployed with GitHub Pages

## Featured projects

- **[Peas on Earth](https://twickss.github.io/peas_on_earth)** — a web GIS app mapping food access and transit equity in Seattle, built with Mapbox GL JS and ArcGIS
- **[Campus Cop](https://campus-cops.web.app)** — a Firebase-deployed campus safety app built and led with a four-person team
- **Digital Age vs. Seattle's Homeless Elderly** — UX research on digital accessibility barriers for unhoused elderly residents
